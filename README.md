# DevOps Mini Projects

This repository contains a collection of hands-on DevOps mini projects completed as part of DevOps practical training. The projects demonstrate core concepts including Linux administration, Docker containerization, Kubernetes orchestration, Terraform infrastructure automation, and CI/CD fundamentals.

## Projects Included

### 1. Linux Sysadmin Automation Kit

**Technologies:** Linux, Bash Scripting

Features:

* Automated system information collection
* Server health monitoring scripts
* Disk usage reporting
* User and uptime information
* Basic automation using shell scripts

Folder:

```text
linux-syadmin-automation/
```

---

### 2. Dockerized To-Do API

**Technologies:** Python Flask, Docker

Features:

* REST API implementation
* Dockerized application deployment
* Container-based execution
* Dependency management using requirements.txt

Folder:

```text
to-do-app-docker/
```

---

### 3. Kubernetes Self-Healing Demo

**Technologies:** Kubernetes, YAML, Kubectl

Features:

* Kubernetes Deployment configuration
* Replica management
* Service configuration
* Pod deletion and self-healing demonstration
* Container orchestration concepts

Folder:

```text
kubernetes-pod-scaling/
```

---

### 4. Terraform Infrastructure Automation

**Technologies:** Terraform, HCL

Features:

* Infrastructure as Code (IaC)
* Terraform configuration files
* Variables and outputs management
* Automated infrastructure provisioning concepts

Folder:

```text
terraform-hands-on/
```

---
